## 📖 About

CUDA Neural Network Framework is an educational deep learning framework written from scratch in **C++** and **CUDA**.

The goal is to understand how modern deep learning frameworks such as **PyTorch** and **TensorFlow** work internally by implementing every major component ourselves.

This project focuses on:

- CUDA Programming
- GPU Memory Management
- Matrix Multiplication
- Neural Network Layers
- Backpropagation
- Optimizers
- Training Engine

No PyTorch Autograd.

No TensorFlow.

Everything is built step-by-step from scratch.

---

# 🎯 Vision

Rather than building a simple MNIST classifier, this project aims to become a lightweight CUDA-based neural network framework suitable for learning AI systems and GPU programming.
